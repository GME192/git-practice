git commands

1. git status: this is used for checking status for git directory
2. git init: It is used for inisialized git repository
3. git add --a: It is used for add all files to moving area
4. git commit -m "": It is used to commit with some message 
5. git log: to show all commit history 
6. rm -rf .git: it will delete your all .git file.  
/dir/ in gitingore: ignore only outer folder 
dir/ ignor all dir files.
